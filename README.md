ShadeSync：開源牙體比色系統 (Open-Source Dental Shade Matching Tool)
專案簡介 (Project Overview)
ShadeSync 是一個致力於解決牙科臨床與數位牙體技術中「比色」難題的開源專案。本專案旨在為全球的牙體技師 (Dental Technicians) 及牙醫師提供一套完全免費且高精度的牙體比色工具。

透過結合普及的智慧型手機相機與強大的 PC 端色彩分析能力，ShadeSync 期望打破傳統高價數位比色儀器的硬體限制，讓每一間牙醫診所與牙體技術所都能輕鬆建立數位化的精準比色標準流程。

核心目標 (Core Objectives)
精準比色： 將牙齒色彩數據化，降低傳統肉眼辨色的主觀誤差與環境光源影響。

開源共享： 軟體完全開源免費，集結全球開發者與牙醫專家的智慧共同優化演算法，推動牙科美學的數位化平權。

降低門檻： 充分利用現有裝置（手機與個人電腦），減少醫療機構在昂貴設備上的投資成本。

系統架構與組成 (System Architecture)
ShadeSync 由兩個主要軟體端點構成，實現從資料採集到深度分析的完整工作流程：

1. Android 應用程式 (資料採集與初步紀錄)
開發語言/框架： Kotlin / Jetpack Compose

主要功能：

影像擷取： 結合 Android 手機的相機，為病患的自然牙或牙體樣品進行高畫質拍攝。

拍攝輔助： 內建對齊線與拍攝指南，協助使用者在標準化的角度與距離下捕捉影像。

即時紀錄與傳輸： 在牙醫診所端快速建立病患的牙齒顏色紀錄，並將影像數據無縫同步或匯出至 PC 端進行後續分析。

2. PC 端 Python 程式 (深度分析與資料庫管理)
開發語言： Python

主要功能：

高階色彩運算： 利用電腦端強大的運算資源，進行複雜的影像處理、環境光源校正與色彩空間（如 RGB 轉換為 CIELAB）轉換。

精準比對演算法： 將擷取到的目標牙齒色彩特徵，與內建的色碼資料庫進行精密比對，計算出最匹配的色號結果。

比色報告生成： 產出視覺化的比色數據報告，協助牙體技師在瓷膏堆疊或染色過程中進行精準的參考。

技術核心：Vita 16 色碼資料庫 (Color Code Database)
專案初期的核心基準，將建立在業界最廣泛使用的 Vita Classical A1-D4® (Vita 16) 實體牙體色卡 之上。

資料庫建置： 團隊將對 Vita 16 的實體色板進行標準化拍攝與色彩特徵萃取，建立一套精準的數位參考色碼庫。

智慧比色： 系統演算法會將臨床拍攝到的牙齒色彩與此 Vita 16 資料庫進行運算，輸出對應的色號（如 A2, B3 等），為牙體技師提供直觀的製作指引。

應用場景 (Application Scenarios)
牙醫診所 (臨床端)： 醫師在為病患進行假牙比色時，不再僅依賴將實體色卡放在牙齒旁用肉眼判斷。可直接使用 Android App 拍照，紀錄客觀的色彩數據，並將檔案傳送給合作的技工所。

牙體技術所 (製作端)： 牙體技師接收數據後，透過 PC 端程式解析出精確的色彩分佈。在假牙製作過程中，也能隨時拍攝半成品，使用系統比對與目標色號的落差，進行微調上色。
