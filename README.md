# ShadeSync

Open-Source Dental Shade Matching Tool for Android and PC

ShadeSync 是一個開源、免費、面向全球牙體技師與牙科團隊的牙體比色工具專案。  
專案目標是以標準牙體色卡（VITA Classical A1-D4, 通稱 Vita 16）建立可驗證的色碼資料庫，並結合 Android 手機相機與 Python 桌面工具，提供一致、可追蹤、可擴充的數位比色流程。

## 專案願景

傳統牙體比色容易受到光源、拍攝設備、顯示器與人眼主觀判斷影響，造成診所與技工所之間的溝通誤差。ShadeSync 希望透過開放資料與開放程式碼，建立一套所有人都能使用、檢視與改進的比色基礎設施。

我們希望達成：

- 降低牙體比色門檻，讓中小型診所與技工所都能使用。
- 提升跨場域（診所/技工所/不同國家）的色彩溝通一致性。
- 建立可持續演進的開源生態，讓研究者與開發者共同優化模型與流程。

## 核心目標

- 建立 Vita 16 標準色階資料庫（含 RGB / CIELAB / metadata）。
- 提供 Android App 進行拍攝、比色、紀錄與比對結果顯示。
- 提供 Python PC 工具進行資料管理、校正、批次分析與報表輸出。
- 設計可追溯的顏色紀錄格式，支援病患歷史顏色追蹤。
- 保持完全免費與開源授權，鼓勵社群參與。

## 第一階段功能範圍（MVP）

### 1) Vita 16 色碼資料庫

- 建立 A1-D4 色階資料表。
- 每個色階至少包含：
  - Shade ID（例如 A1、B2、C3）
  - 參考色彩值（sRGB、CIELAB）
  - 量測與來源資訊（光源條件、白平衡條件、版本）
- 提供資料版本控制，確保後續校正有跡可循。

### 2) Android App

- 使用手機相機拍攝牙齒樣本或色卡。
- 在拍攝流程中提供基本的光源與曝光提示。
- 進行區域取樣與平均色計算。
- 與 Vita 16 資料庫比對後，輸出最接近色階與相似度資訊。
- 建立病患色彩紀錄（時間、條件、結果）。

### 3) Python PC 工具

- 匯入 Android 端紀錄資料。
- 進行色彩空間轉換與誤差分析（例如 Delta E）。
- 支援批次比對、資料修正、統計圖表與報表輸出。
- 支援技工所工作流程中的病例查詢與歷史對照。

## 系統架構（初步）

- `android_app/`：Android 前端與拍攝/比色流程。
- `python/`：桌面端分析工具、資料處理與報表。
- `shared data`（規劃中）：跨平台共用的 shade database 與資料格式。

建議資料流：

1. Android 端取得影像與取樣色值。  
2. 依同版本色碼資料庫完成初步比對。  
3. 將紀錄同步至 PC 端進行深入分析與留存。  
4. 回饋校正結果，持續優化資料庫品質。

## 為什麼開源

- 透明：比色邏輯、校正流程、資料來源可被檢視。
- 可驗證：研究單位與實務端可重現結果。
- 可在地化：不同地區可依設備與流程擴充。
- 可長期維護：不受單一商業產品生命周期限制。

## 開發路線圖（Roadmap）

### Phase 1：資料基礎與 MVP

- Vita 16 基礎資料庫定義與驗證
- Android 拍攝 + 基本比色
- Python 匯入 + 基本報表

### Phase 2：品質提升

- 加入色彩校正流程（白平衡卡 / 光源條件標記）
- 強化 Delta E 評估與結果可信度
- 優化 UI/UX 與跨裝置一致性

### Phase 3：進階能力

- 擴充更多色階系統（例如 Vita 3D-Master）
- 加入模型輔助比色與建議
- 規劃更多平台支援（如 iOS / Web Viewer）
